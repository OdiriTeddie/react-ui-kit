# Security Policy

## Supported Versions

This project is pre-1.0. Security fixes will target the latest published version.

## Reporting a Vulnerability

Please do not open a public issue for a security vulnerability.

Report the issue privately to the maintainers through GitHub security advisories when available, or contact the repository owner directly.

Include:

- A description of the vulnerability.
- Steps to reproduce.
- Affected versions or commits.
- Any known workarounds.
