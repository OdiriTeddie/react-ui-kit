# Code of Conduct

This project follows the Contributor Covenant Code of Conduct.

## Our Standards

Examples of behavior that contributes to a positive environment:

- Using welcoming and inclusive language.
- Being respectful of differing viewpoints and experiences.
- Giving constructive feedback.
- Focusing on what is best for the community.

Examples of unacceptable behavior:

- Harassment, insults, or discriminatory language.
- Publishing private information without permission.
- Sustained disruption of project discussions.

## Enforcement

Project maintainers may remove comments, issues, pull requests, or other contributions that do not align with this Code of Conduct.

Instances of abusive behavior can be reported through the project issue tracker or directly to the maintainers.
